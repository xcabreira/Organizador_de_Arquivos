# 📂 Organizador de Arquivos

Uma automação desenvolvida em Python para facilitar a organização de arquivos e pastas de forma rápida e automática.

## Funcionalidades

- Organiza arquivos por categoria (documentos, imagens, vídeos, etc.);
- Renomeia arquivos automaticamente;
- Move arquivos para pastas específicas;
- Remove arquivos duplicados ou desnecessários;
- Exclui arquivos temporários;
- Gera um relatório das alterações realizadas.

## Tecnologias

- Python 3
- os
- shutil
- pathlib
- hashlib
